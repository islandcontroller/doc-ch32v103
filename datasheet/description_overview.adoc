=== Device overview

[.text-justify]
CH32V103 series devices are offered in three package variants spanning a set of four device models, each with a specific package, memory and peripherals configuration. All available configurations are shown in <<model-comparison>>. The device internal block structure is shown in <<system-diagram>> and a more detailed clock distribution tree is shown in <<clock-tree>>.
